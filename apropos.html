A PROPOS DE MOI

PROJET UNIVERSITAIRE(MASTER)

Dadesso Dadoyi est diplômé d'un Master en Physique : Matériaux, Énergie et Transferts, avec un parcours solide. Il a travaillé sur un projet de caractérisation des matériaux, utilisant la méthode numérique pour résoudre le problème de l'équation de chaleur dans un multimatériau. Grâce à la méthode des éléments finis, nous avons pu résoudre notre problème. Malheureusement, les résultats n'ont pas été vérifiés expérimentalement en raison du manque de laboratoires adéquats dans notre pays, donc les champs de température simulés sont considérés comme des données expérimentales. Le langage Python avec son framework FEniCS nous a permis de simuler les champs de température dans les matériaux. Nous avons identifié les paramètres thermophysiques grâce à un algorithme d'optimisation de machine learning basé sur le modèle des moindres carrés. Cette approche permet de caractériser thermiquement de nouveaux matériaux en utilisant les champs de température mesurés. Ce type de problème est fréquemment rencontré dans les entreprises, où il est crucial de déterminer le matériau adéquat et de prédire la température en temps réel. La connaissance précise de ces paramètres est donc essentielle pour de nombreuses applications industrielles.Exemples des modules suivis : Mécanique des milieux continus, Machine learning, Équations aux dérivées partielles, Simulation de problèmes physiques.
